OculusControllerLeft 

object OculusControllerLeft extends OculusController

Contains the name, event id's and setup method for the left Oculus Controller. All event listeners for inputs to this controller are here.

Can be thought of as a mapping from a subset of the available inputs to interactions (see plots.Interactions).
