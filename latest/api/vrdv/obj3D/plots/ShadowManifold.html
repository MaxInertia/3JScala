ShadowManifold 

Companion object ShadowManifold

case class ShadowManifold(data: Data, tau: Int, scp: ScaleCenterProperties, points: Points) extends Plot3D with Product with Serializable

A reconstruction of an attractor manifold generated from data on a single variable. Each point can be thought of as the history of the variable over some interval of time.
